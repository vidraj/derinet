This directory contains Czech mnd SLovak segmentation files.
Currently, it includes:

Czech segmentation data extracted from RMSS (rmss_data_cz.tsv, RMSS-CZ)
Slovak segmentation data from RMSS (rmss_data_sk.tsv, RMSS-SK)
Manually segmented lemmas from Zdeněk Žabokrtský (man_cz.tsv, MANR)
Data for Sigmorphon 2023 (sigmorphon_cz.tsv, SIG_...)
Czech data from retrogade dictionary of Eleanora Slavíčková(slav_ncl_cz.tsv)

In total:
76117 unique Czech segmentations
72161 unique Czech words (regardless of segmentation)
